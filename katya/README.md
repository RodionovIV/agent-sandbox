# Katya: LLM Агент

## Описание
Проект Katya представляет собой реализацию LLM-агента, способного взаимодействовать с внешними системами и выполнять задачи.

## Структура проекта
- `agents/`: Директория с реализацией агента.
- `core/`: Директория с реализацией ядра проекта.
- `data/`: Директория с данными.
- `instructions/`: Директория с инструкциями.
- `tests/`: Директория с тестами.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите проект: `python run.py`

## Тестирование
Для тестирования проекта используйте файл `tests/test_agent.py`.

## Документация
Документация по проекту доступна в файле `README.md`.
