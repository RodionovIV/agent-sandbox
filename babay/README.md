# AI помощник для изучения иностранных языков\n\n## Описание\nПроект представляет собой AI помощника для изучения иностранных языков, реализованный с использованием FastAPI, LangChain и GigaChat.\n\n## Структура проекта\n- `main.py`: точка входа в приложение.\n- `api/endpoints.py`: FastAPI роутеры.\n- `models/agent.py`: Pydantic модели для API.\n- `services/agent_system.py`: логика агента.\n- `dependencies/`: дополнительные зависимости (если требуются).\n\n## Установка\n1. Установите зависимости:\n```bash\npip install -r requirements.txt\n```\n\n2. Запустите приложение:\n```bash\nuvicorn main:app --reload\n```\n\n## Использование\nОтправьте POST запрос на `/language-learning` с телом запроса:\n```json\n{\n    "query": "Ваш запрос"\n}\n```\n\n## Пример ответа\n```json\n{\n    "result": "Ответ на ваш запрос"\n}\n```\n\n## Тестирование\nПроведите тестирование нового функционала и отладьте возможные ошибки.\n\n## Автор\nВаше имя
